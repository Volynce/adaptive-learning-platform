\# Step-03: Каркас приложения FastAPI (src + /api/v1)



\## Решения

1\) Введён src-based каркас пакета adaptive\_learning\_platform.

2\) Точка входа: main.py с фабрикой create\_app() и экспортом app.

3\) Версионирование API: префикс /api/v1 через централизованные роутеры.

4\) Подключены модульные роутеры (пока заглушки health), чтобы зафиксировать границы модулей.



\## Зачем

\- Обеспечить единообразный запуск и тестирование приложения.

\- Зафиксировать структуру и контракт маршрутизации до внедрения БД и доменной логики.



\## Критерий готовности

\- poetry run uvicorn ... запускается.

\- Swagger показывает /api/v1/\*/health.
