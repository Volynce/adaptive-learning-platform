\# Step-02: Poetry + Python 3.12 (зафиксировано)



\## Решения

1\) Менеджер зависимостей: Poetry (pyproject.toml + poetry.lock).

2\) Версия Python проекта: 3.12.x (ограничение: >=3.12,<3.13).

3\) Установка зависимостей на этом шаге выполняется командой:

&nbsp;  poetry install --no-root

&nbsp;  Причина: src-каркас и пакет adaptive\_learning\_platform будут созданы на следующем шаге (Step-03).



\## Зачем

\- Обеспечить воспроизводимость окружения: одинаковые зависимости и версии на любом git checkout.

\- Исключить проблемы совместимости библиотек с Python 3.13.

\- Подготовить основу для следующих шагов: src-структура, create\_app(), /api/v1, конфигурация env.



\## Критерий готовности

\- pyproject.toml и poetry.lock присутствуют в репозитории.

\- poetry env использует Python 3.12.x.

\- poetry install --no-root выполняется без ошибок.
